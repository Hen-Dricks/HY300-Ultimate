# Annexes

Les annexes regroupent l'ensemble des informations de référence utilisées tout au long de l'étude.

Contrairement aux chapitres principaux, elles sont organisées comme un manuel pratique destiné à accompagner les expérimentations.

## Contenu

| Fichier                    | Description                                                      |
| -------------------------- | ---------------------------------------------------------------- |
| a-commandes-reseau.md      | Commandes réseau Android et Linux utilisées pendant l'étude      |
| b-commandes-partitions.md  | Commandes de manipulation des partitions et des images système   |
| c-commandes-docker.md      | Commandes Docker utilisées pour l'environnement de développement |
| d-hashes-de-reference.md   | Empreintes cryptographiques des principaux artefacts             |
| e-erreurs-et-solutions.md  | Erreurs rencontrées et procédures de résolution                  |
| f-glossaire.md             | Définitions des principaux termes techniques                     |
| g-checklist-publication.md | Liste complète des vérifications avant publication d'une build   |

Ces annexes évolueront au fur et à mesure du développement du projet.
